# Laptop Rental Landing Page

![Hero Preview](./public/image.png)

A modern and responsive laptop rental landing page built with Next.js, Tailwind CSS, and Framer Motion.

## Tech Stack

- Next.js
- Tailwind CSS
- Framer Motion
- Lenis

## Features

- Responsive layout
- Clean digital agency design
- Scroll-based animations
- Structured motion system
